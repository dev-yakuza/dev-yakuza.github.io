## Outline

I use Jekyll and GitHub to write this blog post. I was looking for a way to display the number of visitors on the static site like this, and I found `HITS` project.

- HITS: [GitHub](https://github.com/dwyl/hits){:rel="nofollow noreferrer" target="_blank"}

## What is HITS

HITS is originally the project to show visitors count for GitHub project.

![How to show number of visitors via HITS - HITS for github](/assets/images/category/share/2020/hit-counts/hits-for-github.jpg)

To use HITS, visit the site below,

- HITS: [http://hits.dwyl.io/](http://hits.dwyl.io/){:rel="nofollow noreferrer" target="_blank"}

You can see the screen like below.

![How to show number of visitors via HITS - HITS service](/assets/images/category/share/2020/hit-counts/hits-service.jpg)

On above screen, when you insert `username` and `repo/project` of GitHub,

![How to show number of visitors via HITS - username, repo/project](/assets/images/category/share/2020/hit-counts/hits-username-repo.jpg)

You can get the URL like below.

```md
{% raw%}
![HitCount](http://hits.dwyl.com/dev-yakuza/react-native-image-modal.svg)
{% endraw %}
```

If you insert this link to README on Github, you can display visitors count.

- Visitors count: ![HitCount](http://hits.dwyl.com/dev-yakuza.github.io{{ page.url }}-example.svg)

If you refresh, you can see the number is changed dynamically.

## Jekyll blog

I use `HITS` to show the number of visitors on Jekyll blog.

![How to show number of visitors via HITS - hits on jekyll](/assets/images/category/share/2020/hit-counts/hits-on-jekyll.jpg)

To show visitors count like above, I use the code below on `_layouts/post.html`.

```html
{% raw%}
<img style="width: inherit;" src="http://hits.dwyl.com/dev-yakuza.github.io{{ page.url }}.svg" alt="hit count image"/>
{% endraw %}
```
