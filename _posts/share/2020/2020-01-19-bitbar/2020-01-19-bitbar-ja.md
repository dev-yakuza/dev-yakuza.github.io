## 概要

最近Githubの1日コミットをしています。1日コミット内容を確認するため、ChromeへGithubプロフィルページをいつも開いて、毎回確認していました。このような問題を解決するため`BitBar`と言うツールと`github-contribution`プラグインを使ってGithubのコミット内容をMacのStatusbarへ表示するようにしてみます。

![1日コミットを維持するためBitBarの活用 - bitbar github-contribution](/assets/images/category/share/2020/bitbar/bitbar-github-contribution.jpg)

- 公式サイト: [BitBar](https://getbitbar.com/){:rel="nofollow noreferrer" target="_blank"}
- github-contributionプラグイン: [github-contribution](https://getbitbar.com/plugins/Dev/GitHub/github-contribution.10m.rb){:rel="nofollow noreferrer" target="_blank"}

## BitBarインストール

下記のコマンドでBitBarをインストールします。

```bash
brew cask install bitbar
```

インストールが完了されたら、下記のようにアプリリストへ追加されたことが確認できます。インストール完了されたBitBarアプリを実行します。

BitBarアプリを実行すると下記のような画面が見えます。`Open`を押してアプリを実行します。

![1日コミットを維持するためBitBarの活用 - BitBar実行](/assets/images/category/share/2020/bitbar/open-bitbar-app.jpg)

もし、プラグインのフォルダを選択する画面が出ると上で準備したプラグインのフォルダを選択します。プラグインのフォルダは下記のようにいつでも変更できます。

## プラグインのインストール

下記のリンクをクリックしてgithub-contributionプラグインのサイトへ移動します。

![1日コミットを維持するためBitBarの活用 - open bitbar](/assets/images/category/share/2020/bitbar/open-bitbar.jpg)

## プラグインソースコード追加

プラグインへ必要なソースコードを追加する必要があります。上で作ったプラグインフォルダへ移動した後、`github-contribution.10m.rb`ファイルを生成します。

![1日コミットを維持するためBitBarの活用 - github contribution plugin site](/assets/images/category/share/2020/bitbar/github-contribution-plugin-site.jpg)

ファイルを生成したら、プラグインサイトの`+ Add to BitBar`横にある`View source`リンクをクリックします。

![1日コミットを維持するためBitBarの活用 - github contribution plugin sourcecode](/assets/images/category/share/2020/bitbar/github-contribution-plugin-source.jpg)

該当ソースコードをコピーして、上で作った`github-contribution.10m.rb`ファイルへペーストします。

## bitbarrc設定

インストールが終わったら、Githubのアカウントを設定する必要があります。自分のエディタをで`~/.bitbarrc`を開きます。

```bash
# vscode
code ~/.bitbarrc
```

そして、下記の内容を追加します。

```bash
[github_contribution]
username = <GITHUB_USERNAME>
max_contributions = 10
```

ここで`<GITHUB_USERNAME>`を自分のGithubの名前で変更します。

## プラグインの実行

今までインストールや設定したプラグインを実行してみましょう。Macの右上にあるStatusbarを選択して`Refresh all`を選択します。

![1日コミットを維持するためBitBarの活用 - bitbar refresh all](/assets/images/category/share/2020/bitbar/bitbar-refresh-all.jpg)

そしたら下記のようにBitBarのgithub-contributionが実行されて現在のコミット状態が確認できます。

![1日コミットを維持するためBitBarの活用 - bitbar github-contributionインストール完了](/assets/images/category/share/2020/bitbar/bitbar-github-contribution.jpg)

BitBarのgithub-contributionは10分ことに情報を更新します。

## 完了

これで1日コミットを維持するため、BitBarをインストールしてgithub-contributionプラグインを設定してみました。

今から毎回Githubのプロフィールページで確認しなくても1日コミットしたか確認できるようになりました。じゃ、頑張って1日コミットをやってみましょう！
