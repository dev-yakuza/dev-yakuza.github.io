## 개요

이번 블로그 포스트에서는 Rails의 Controller에서 사용할 수 있는 `head` 메서드가 무엇이며, 어떻게 사용하는지에 대해서 알아보려고 합니다.

## head

head 메서드는 Rails에서 Controller에서 사용되며, HTTP 상태만 응답할 때 사용됩니다.

head 메서드는 다음과 같이 사용합니다.

```ruby
head status
```

- status: HTTP의 응답 상태

예를 들면, head 메서드 다음과 같이 Rails의 Controller에서 사용할 수 있습니다.

```ruby
class BooksController < ApplicationController
  def show
    return head :not_found if id != 1
    ...
  end
end
```

head 메서드를 사용하면 HTTP 상태만 응답하고 내용물은 별로도 출력하지 않습니다. 위에 예제에서는 `id`가 `1`이 아닌 경우에 `:not_found`, 즉 `404 Not Found`를 응답합니다. 이렇게 HTTP의 응답과 함께 별도로 내용물을 전달할 필요가 없을 때, `head` 메서드를 사용합니다.
