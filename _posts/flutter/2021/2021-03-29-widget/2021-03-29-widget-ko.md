## 프로젝트 생성

그럼 이제 Flutter의 새로운 프로젝트를 생성해 봅시다. 다음 명령어를 사용하여 Flutter의 새로운 프로젝트를 생성합니다.

```bash
# flutter create [Project Name]
flutter create first_app
```

프로젝트명은 소문자와 `_`를 사용하는 `스네이크 케이스(Snake case)`를 사용해야 합니다.

## AppBar

```dart
AppBar(
  title: Text('HOME'),
  centerTitle: true, // 안드로이드에서 앱 타이틀 가운데 정렬
  backgroundColor: Colors.blue,
  elevation: 0.0, // 하단 그림자 없애기 > flat으로 만들기
)
```

## Column

```dart
Column(
  mainAxisAlignment: MainAxisAlignment.center// 세로축 상, 중, 하
```
