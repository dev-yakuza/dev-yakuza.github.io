  • Babel ES6/ES7
  • Bracket Pair Colorizer

  • IntelliSense for CSS, SCSS class names in HTML, Slim and SCSS
  • Node.js Modlues Intellisense
  • npm Intellisense
  • Path Intellisense
  • PHP Intellisense
  • Prettier - Code Formatter
  • Python
  • Trailing Spaces
  • TSLint
  • vscode-icons
  • vscode-styled-components
