create-react-appлҸ„ мқҙл ҮкІҢ TypeScriptм—җ мӨ‘мҡ”м„ұмқ„ мқём§Җн•ҳкі  мһҲкі , TypeScriptлҘј лҚ”мҡұ к°„лӢЁн•ҳкІҢ м§Җмӣҗн•ҳкё° мң„н•ҙ Template мҳөм…ҳмқ„ м ңкіөн•ҳкі  мһҲмҠөлӢҲлӢӨ. к·ёлҹј create-react-appмқҳ Tempate мҳөм…ҳмқ„ мӮ¬мҡ©н•ҳм—¬ Reactмқҳ TypeScript н”„лЎңм қнҠёлҘј мғқм„ұн•ҙ ліҙлҸ„лЎқ н•©мӢңлӢӨ.

лӢӨмқҢ лӘ…л №м–ҙлҘј мӢӨн–үн•ҳм—¬ Reactм—җ TypeScriptк°Җ м Ғмҡ©лҗң н”„лЎңм қнҠёлҘј мғқм„ұн•©лӢҲлӢӨ.
