설치

jekyll은 Mac, Windows, Ubuntu에 설치할 수 있다. 우리는 주로 Mac과 Windows를 사용하기 때문에 여기에는 Mac과 Windows에 설치하는 방법만 소개하겠다.

Mac과 windows는 Ruby를 설치하는 방법만 다를 뿐, 다른 부분은 동일하다. 자세한 내용은 jekyll 홈페이지에서 확인하길 바란다.

사용 버전

현재 블로그를 작성하는 시점에 사용하는 버전은 아래와 같다.

  • Ruby: ruby 2.5.1p57
  • jekyll: jekyll 3.7.3

Ruby 설치

jekyll을 하나의 Ruby Gem이므로, Ruby를 설치해야한다.

Mac에 Ruby 설치

brew install ruby

Windows에 Ruby 설치

  • Ruby installer를 사용하자.
  • 다운로드 사이트: RubyInstaller for Windows
  • Ruby + Devkit의 2.4이상의 버전을 다운받아 설치하자.

Ruby 설치 확인

ruby -v

jekyll Ruby Gem 설치

Ruby를 설치했다면, Ruby를 사용하여 jekyll Gem을 설치하자.

jekyll Ruby Gem 설치

gem install bundler jekyll

jekyll Ruby Gem 설치 확인

jekyll -v

jekyll Gem을 이용한 기본 프로젝트 생성

프로젝트 생성

jekyll new [프로젝트명] 명령어를 이용하여 jekyll 프로젝트 생성

jekyll new dev-yakuza.github.io

프로젝트 폴더 확인

// cd dev-yakuza.github.io

|-- _posts
|   |-- 2018-09-08-welcome-to-jekyll.markdown
|-- .gitignore
|-- 404.html
|-- about.md
|-- index.md
|-- Gemfile
|-- Gemfile.lock

jekyll 시작하기

bundle exec jekyll serve

브라우저에서 http://127.0.0.1:4000 접속하여 확인하자

에러 대응

새로운 맥(Mac)에 기존의 jekyll 블로그를 설정하기 위해 jekyll을 설치하고 gem install으로 필요한 파일을 설치하던 중 에러가 발생했습니다. 사용하는 플러그인중 자바가 필요한 플러그인이 있었습니다. 자바를 설치한 후 문제없이 플로그인들을 설치할 수 있었습니다. 혹시 플러그인 설치중 에러가 나시는 분들은 자바가 설치되어 있는지 확인해 보시기 바랍니다.
